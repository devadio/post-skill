AI Agent Token

Use this token for the Google Sheet web-app API. Read access is open. Write access needs this token.

Short Tips

  • Use this token only for AI-agent write requests.
  • The API is limited to the post tab and range A:I.
  • Row 1 headers are protected and cannot be changed.
  • If a token is exposed, use Reset Token immediately.
  • The AI agent should mainly write content fields and avoid overwriting helper or output fields unless intended.
